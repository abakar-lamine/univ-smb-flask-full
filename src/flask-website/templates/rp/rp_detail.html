Fichier Nginx (Reverse Proxy)

server {
    listen {{ rp.ip_bind }}:80;
    server_name {{ rp.name | lower | replace(' ', '-') }}.univ-savoie.fr;

    access_log /var/log/nginx/{{ rp.name | lower }}_access.log;

    location / {
        proxy_pass {{ rp.target }};
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
🚀 Préparation au déploiement
Méthode 1 : Docker
docker run -d --name nginx-rp-{{ rp.id }} \
  -p 80:80 \
  -v $(pwd)/nginx_{{ rp.name }}.conf:/etc/nginx/conf.d/default.conf:ro \
  nginx:alpine
Méthode 2 : Native (Linux)
sudo cp nginx_{{ rp.name }}.conf /etc/nginx/conf.d/
sudo nginx -t && sudo systemctl restart nginx
