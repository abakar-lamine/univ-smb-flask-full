Fichier Nginx (Web Server)

server {
    listen {{ ws.port }};
    server_name {{ ws.name | lower | replace(' ', '_') }}.local;

    root {{ ws.root_path }};
    index index.html index.htm;

    access_log /var/log/nginx/{{ ws.name | lower | replace(' ', '_') }}_access.log;
    error_log /var/log/nginx/{{ ws.name | lower | replace(' ', '_') }}_error.log;

    location / {
        try_files $uri $uri/ =404;
    }
}
🚀 Préparation au déploiement
Méthode 1 : Docker
# Lier le port local au port Nginx, et monter les fichiers HTML
docker run -d --name nginx-ws-{{ ws.id }} \
  -p {{ ws.port }}:{{ ws.port }} \
  -v $(pwd)/nginx_{{ ws.name }}.conf:/etc/nginx/conf.d/default.conf:ro \
  -v {{ ws.root_path }}:/usr/share/nginx/html:ro \
  nginx:alpine
Méthode 2 : Native (Linux)
sudo cp nginx_{{ ws.name }}.conf /etc/nginx/conf.d/
# S'assurer que le dossier web existe
sudo mkdir -p {{ ws.root_path }}
sudo nginx -t && sudo systemctl restart nginx
